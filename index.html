potrzebujesz

Jedno narzędzie do zarządzania kontami Roblox.

🌐
Logowanie przez przeglądarkę

Wbudowana przeglądarka — logujesz się normalnie na roblox.com. Sesja zapisana na stałe, bez wklejania cookie.

Electron Session
🎮
Uruchamianie gier

Dołącz do dowolnej gry po Place ID lub z listy ulubionych — konto wymuszone automatycznie przez ticket.

Auth Ticket API
🔗
Prywatne serwery

Dołącz przez link Share lub privateServerLinkCode. Ulubione serwery zapisane — jeden klik i grasz.

Share Links
📊
Statusy kont

Sprawdź w czasie rzeczywistym czy konto jest online, offline, lub w jakiej grze się aktualnie znajduje.

Presence API
⚙️
FPS i jakość grafiki

Ustaw limit FPS (30–240) i poziom grafiki bezpośrednio w pliku XML Roblox. Działa po restarcie gry.

GlobalBasicSettings.xml
⭐
Ulubione gry i serwery

Zapisz ulubione gry i prywatne serwery. Miniaturka i nazwa pobierane automatycznie z API Roblox.

Favorites
✏️
Prefixy kont

Oznacz każde konto własnym prefiksem np. [FARM], [MAIN] dla lepszej organizacji wielu kont.

Labeling
🔒
Bezpieczeństwo

Sesje szyfrowane lokalnie. Żadne dane nie opuszczają Twojego komputera. Brak zewnętrznych serwerów.

AES Encrypted

Animowany splash screen z płynnym wjazdem nazwy i autora przy każdym uruchomieniu aplikacji.
